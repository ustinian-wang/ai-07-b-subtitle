# 使用指南

面向**第一次使用**的读者：按界面走一遍即可。

## 界面布局

```
┌─────────────┬──────────────────┬─────────────┐
│  笔记库侧栏  │   笔记提取 / 详情   │  对话助手    │
│  文件夹树    │   粘贴 URL 提取     │  @ 引用笔记  │
│  批量操作    │   查看字幕/正文     │  SSE 流式    │
└─────────────┴──────────────────┴─────────────┘
│ 设置（左下角）                                              │
└────────────────────────────────────────────────────────────┘
```

- **前端地址**：http://localhost:9177
- 数据保存在本机 `backend/data/`，换电脑需自行备份

## 提取笔记

### B 站

1. 复制视频链接（`bilibili.com/video/BV…`、`b23.tv` 短链等）
2. 粘贴到中间输入框，分 P 默认 1
3. 点 **提取**

成功后在侧栏出现笔记；详情区显示带时间轴字幕。

**需登录字幕**：部分视频要配 B 站 `SESSDATA`，见 [配置说明 · B 站](configuration.md#b-站-sessdata)。

### 小红书

1. 粘贴 `xiaohongshu.com/explore/…` 或 `xhslink.com` 链接
2. 点 **提取**

需配小红书 Cookie，见 [配置说明 · 小红书](configuration.md#小红书-cookie)。

### 去重

同一视频（B 站：`bvid + 分P + 字幕轨`；小红书：`note_id`）已存在时：

- 提示 **直接查看已有笔记** 或 **重新提取**（`force`）
- 不会重复抓取，除非选重新提取

## 笔记库

### 打开笔记

侧栏点击笔记标题 → 中间显示详情（须看到 **「已保存 #xxx」** 才表示已入库）。

### 文件夹

- **新建文件夹**：侧栏按钮，可选父级目录
- **移动笔记**：勾选后点「移动」，或拖到目标文件夹
- **移动文件夹**：拖拽文件夹到另一文件夹（「全部」= 根目录）

### 批量操作

先勾选笔记（或框选），再使用：

| 按钮 | 作用 |
|------|------|
| 移动 | 改文件夹 |
| 复制 | 纯文本到剪贴板 |
| 导出 txt/json | 下载文件 |
| 同步 Notion | 推到 Notion（需配置） |
| 删除 | 删除本地记录 |

## 对话助手

1. 在 [配置说明 · LLM](configuration.md#对话-llmopenai-兼容) 配好 API
2. 右侧新建或选择会话
3. 输入问题；用 **`@`** 选笔记，或从侧栏 **拖拽** 笔记到对话区作上下文
4. 发送后 SSE 流式返回

单条引用笔记正文上限约 12000 字符（后端注入 system prompt）。

## 同步 Notion

**前置**：完成 [Notion 配置](configuration.md#notion-同步)。

1. **单条**：打开已保存笔记 → 详情 toolbar **「同步到 Notion」**
2. **批量**：侧栏勾选 → **「同步 Notion」**

规则：

- 在配置的**父页面**下创建子页面
- **标题相同**则覆盖正文（非新建 duplicate 页）
- B 站分 P>1 时标题带 `(P2)` 等后缀

按钮灰色常见原因：

- 单条：笔记未保存（无 `record_id`）→ 从侧栏打开已入库笔记
- 批量：未勾选任何笔记

## Chrome 扩展：B 站一键导入

可选：安装 Chrome 扩展 **chromematools**（与本服务配套，需单独安装扩展包）。

1. 本服务在 **8907** 端口运行
2. 设置页配好 **B 站 SESSDATA**（如需登录字幕）
3. `chrome://extensions` 加载扩展并刷新
4. 打开 B 站视频页 → 右下角 **📝** → 导入到「**B站资源**」文件夹

扩展通过 `background.js` 代理 `http://127.0.0.1:8907`，避免 HTTPS 页面 Mixed Content。

## 故障排查

| 现象 | 处理 |
|------|------|
| 提取失败 / 需登录 | 设置页更新 B 站 Cookie |
| 对话无响应 | 检查 OpenAI Key、Base URL、Model |
| Notion 404 | 父页面是否已 **Connections → 分享给 Integration** |
| 同步按钮灰 | 从侧栏打开已保存笔记，或先勾选笔记 |
| 扩展导入失败 | 确认 `curl http://127.0.0.1:8907/api/health` 正常 |

更多接口错误见 [API 参考](api.md)。
